# deepsearcher-demo

手动实现deepsearcher核心逻辑：
向量构建 + 语义检索 + 大模型问答

## 使用方式

```bash
pip install -r requirements.txt
export OPENAI_API_KEY=你的API_KEY
python main.py
```
